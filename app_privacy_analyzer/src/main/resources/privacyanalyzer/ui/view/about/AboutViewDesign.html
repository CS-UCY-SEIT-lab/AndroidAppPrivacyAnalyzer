About
App Privacy Analyzer is a web application that analyzes android applications (APKs) for privacy and security issues. You can find a list with all analyzed APKs and their details, but also you can upload your APK and see the results. The results of your APK will be saved in our list, so anyone can access them.

The analysis system combines many popular static analysis tools such as Androguard, LibRadar, RiskInDroid, PScout to provide the analysis results.

Each analysis contains:

Basic information: application name, file hash, package name, SDK version, etc.
Permission information: Extract permissions and detect if the declared permission is used or not.
Classification of application as dangerous or not.
Calculation of risk score of the application.

The web application has been developed for my Diploma Project at University of Cyprus.
